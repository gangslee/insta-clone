# Insta-Clone

Instagram clone with Express + Prisma + React and React Native

## User Stories

- [ ] See the feed
- [ ] Like a photo
- [ ] Comment on a photo
- [ ] Search by user
- [ ] Search by location
- [ ] See user profile
- [ ] Follow / Unfollow User
- [ ] See the full photo
- [ ] Edit my profile
- [ ] Create account
- [ ] Log in
- [ ] Upload a photo
- [ ] Edit the photo (Delete)
